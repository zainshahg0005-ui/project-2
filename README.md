
# To Do or Not To Do App

## Project Description

This project is a simple To-Do web application built using HTML and CSS. It displays a weekly task planner from Monday to Friday along with a reminders section. The layout is designed using CSS Flexbox to demonstrate alignment, spacing, and responsive design.

## Setup / Run Instructions

1. Download or clone the repository.
2. Open the project folder.
3. Run the application by opening the `index.html` file in any web browser.
